# vue-scroll-nav

## 描述
此为vue版滚动导航组件，实现
- 滚动时，根据滚动距离判断当前内容所属哪个导航
- 点击导航，页面滚动到内容对应位置
- 可控制导航栏是否吸附固定，如吸顶效果，即滚动一定距离后，导航栏固定起来，不会被滚动出可视区域
- 可用在页面任何位置上，不一定是顶部导航这种简单应用哦，可以复杂型应用
